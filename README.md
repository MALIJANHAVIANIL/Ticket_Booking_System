# 🎟️ Ticket Booking System (Java)

## 📌 Project Description

✨ A simple **Ticket Booking System** developed using **Core Java and Collection Framework**.
This system allows users to:

* 🎫 Book tickets
* 🔍 Check booking status
* 💺 Select and verify seat availability

---

## 🚀 Features

✅ Book Ticket with Name & Seat Number
❌ Prevent Duplicate Registration
❌ Prevent Same Seat Booking
🔍 Check Status by:

* 👤 Name
* 🎟 Ticket Number
* 💺 Seat Number

---

## 🛠️ Technologies Used

* ☕ Core Java
* 📦 Collections (`HashMap`, `HashSet`)
* 🧠 OOP Concepts
* ⌨️ Scanner (User Input Handling)

---

## 🧩 Data Structures Used

| Data Structure | Purpose               |
| -------------- | --------------------- |
| `HashMap`      | Store Ticket → Name   |
| `HashMap`      | Store Ticket → Seat   |
| `HashSet`      | Avoid duplicate users |
| `HashSet`      | Track booked seats    |

---

## 🔄 Project Flow

```text
Start
  ↓
Menu (1.Book 2.Status 3.Exit)
  ↓
User Input
  ↓
✔ Booking OR ✔ Status Check
  ↓
Display Result
  ↓
End / Repeat
```
